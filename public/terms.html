2 Descripcion del Servicio

WhatsApp Sender Pro es una plataforma tecnologica que permite a los usuarios enviar mensajes a traves de WhatsApp Web utilizando la biblioteca de codigo abierto Baileys. El Servicio incluye:

  • Conexion a WhatsApp Web mediante codigo QR
  • Gestion de contactos y listas de destinatarios
  • Creacion y uso de plantillas de mensajes
  • Envio de mensajes de texto, imagenes y archivos de audio
  • Panel de control con estadisticas de envio
  • Programacion y gestion de campanas de mensajeria
  • Chatbot y sistema de respuestas automatizadas con flujos personalizables
  • Integracion opcional con proveedores de inteligencia artificial (OpenAI y otros)
  • Bandeja de entrada para visualizar y responder mensajes recibidos de contactos
  • Sistema de planes y suscripciones con acceso a la API publica

Politica de Privacidad

Como recopilamos, usamos y protegemos su informacion

10 Recopilacion de Datos

Recopilamos la siguiente informacion cuando usted utiliza la Plataforma:

10.1 Datos proporcionados por el Usuario

  • Datos de registro: Nombre, direccion de correo electronico
  • Datos de conexion: Numero de telefono de WhatsApp vinculado
  • Datos de uso: Contactos importados, plantillas de mensajes creadas, metadatos de campanas (nombre, fecha, numero de destinatarios)
  • Datos de chatbot: Configuracion de flujos de chatbot, mensajes de bienvenida, prompts del sistema de IA
  • Claves de API de IA: Almacenadas de forma cifrada (AES-256) para la integracion con proveedores de IA
  • Mensajes entrantes: Mensajes recibidos de contactos de WhatsApp para la funcionalidad de bandeja de entrada

10.2 Datos que NO recopilamos

  • NO almacenamos el contenido de los mensajes salientes una vez entregados (los mensajes entrantes se almacenan temporalmente en la bandeja de entrada hasta que el usuario los elimine)
  • NO leemos ni accedemos a las conversaciones de WhatsApp del usuario fuera de las funcionalidades de bandeja de entrada y chatbot configuradas por el propio usuario
  • NO recopilamos datos de los destinatarios de los mensajes mas alla del numero de telefono y nombre proporcionado por el usuario o recibido por WhatsApp
  • NO recopilamos datos de ubicacion, grabaciones de voz ni datos biometricos

11 Almacenamiento de Datos

Los datos del Usuario se almacenan utilizando los siguientes servicios y tecnologias:

  • Autenticacion: Gestionada a traves de Firebase Authentication (Google). Las credenciales son procesadas y almacenadas por Google segun sus propias politicas de seguridad.
  • Perfiles de usuario: Almacenados en Google Cloud Firestore, con las medidas de seguridad proporcionadas por Google Cloud Platform.
  • Contactos y metadatos de campanas: Almacenados en bases de datos PostgreSQL protegidas, con acceso restringido y cifrado en transito.
  • Sesiones de WhatsApp: Los datos de sesion se almacenan temporalmente en Redis con cifrado y se eliminan al desconectarse.
  • Configuracion de chatbot: Los flujos, nodos y configuraciones del chatbot se almacenan en PostgreSQL asociados a la cuenta del usuario.
  • Claves de API de IA: Las claves de API de proveedores de IA se almacenan cifradas con AES-256 en PostgreSQL. Solo se descifran en memoria al momento de realizar llamadas al proveedor de IA.
  • Mensajes entrantes: Los mensajes recibidos de contactos de WhatsApp se almacenan en PostgreSQL para la funcionalidad de bandeja de entrada.
  • Conversaciones de chatbot: El estado de las conversaciones del chatbot (nodo actual, contexto) se almacena en PostgreSQL para mantener la continuidad del flujo.

12 Comparticion de Datos

WhatsApp Sender Pro NO vende, alquila ni comparte datos personales de los usuarios con terceros con fines comerciales, publicitarios o de marketing.

Los datos pueden compartirse unicamente en los siguientes casos:

  • Proveedores de infraestructura: Los datos se procesan a traves de servicios de terceros (Google Cloud, Firebase) necesarios para el funcionamiento de la Plataforma, sujetos a sus respectivas politicas de privacidad y acuerdos de procesamiento de datos.
  • Requerimientos legales: Cuando sea necesario para cumplir con una obligacion legal, orden judicial o requerimiento de autoridad competente.
  • Proteccion de derechos: Cuando sea razonablemente necesario para proteger los derechos, propiedad o seguridad de WhatsApp Sender Pro, sus usuarios o el publico.
  • Proveedores de inteligencia artificial: Cuando el Usuario habilita la funcion de IA en el chatbot, los mensajes entrantes de sus contactos se envian al proveedor de IA seleccionado (por ejemplo, OpenAI) utilizando la clave de API del propio Usuario. Este envio es necesario para generar respuestas inteligentes y esta sujeto a los terminos del proveedor de IA.

13 Seguridad de los Datos

Implementamos medidas de seguridad tecnicas y organizativas para proteger los datos del usuario:

  • Cifrado en transito: Todas las comunicaciones se realizan a traves de conexiones seguras HTTPS/TLS.
  • Autenticacion robusta: Utilizamos JSON Web Tokens (JWT) para la autenticacion y autorizacion de las sesiones.
  • Gestion de sesiones: Control de sesion unica activa por cuenta, con tokens de sesion revocables.
  • Acceso restringido: Los datos se almacenan en servidores con acceso limitado y monitorizado.
  • Verificacion de correo: Requerimos verificacion de correo electronico para activar las cuentas.
  • Cifrado de claves de API: Las claves de API de proveedores de IA se almacenan cifradas con AES-256-CBC y solo se descifran en memoria durante el procesamiento.

A pesar de estas medidas, ningun sistema de seguridad es infalible. No podemos garantizar la seguridad absoluta de los datos, aunque nos comprometemos a notificar a los usuarios afectados en caso de una brecha de seguridad significativa.

14 Retencion de Datos

  • Datos de cuenta: Se conservan mientras la cuenta permanezca activa.
  • Contactos y plantillas: Se mantienen mientras la cuenta este activa. El usuario puede eliminarlos manualmente en cualquier momento.
  • Estadisticas y metadatos: Se conservan por un periodo maximo de 12 meses con fines analiticos.
  • Al cancelar la cuenta: Todos los datos personales, contactos, plantillas y metadatos asociados se eliminan dentro de un plazo de 30 dias habiles.
  • Datos de sesion: Los datos temporales de sesion de WhatsApp se eliminan automaticamente al desconectarse o tras un periodo de inactividad.
  • Mensajes de bandeja de entrada: Los mensajes entrantes se conservan hasta que el Usuario los elimine manualmente. El Usuario puede eliminar conversaciones individuales o todo el historial.
  • Conversaciones de chatbot: Los estados de conversacion del chatbot se reinician automaticamente tras periodos de inactividad configurables.
  • Claves de API de IA: Se conservan cifradas mientras la funcion de IA este habilitada. Se eliminan al desactivar la integracion de IA o al cancelar la cuenta.

15 Bandeja de Entrada y Mensajes Recibidos

La Plataforma almacena los mensajes entrantes recibidos de los contactos de WhatsApp del Usuario para proporcionar la funcionalidad de bandeja de entrada.

15.1 Almacenamiento de Mensajes

  • Los mensajes entrantes (texto, tipo de mensaje y metadatos basicos) se almacenan en nuestra base de datos asociados a la cuenta del Usuario.
  • Los mensajes enviados como respuesta desde la bandeja de entrada tambien se registran en el historial de la conversacion.
  • Las respuestas automaticas del chatbot se identifican claramente como tales en el historial.

15.2 Control del Usuario

  • El Usuario puede eliminar conversaciones completas de un contacto en cualquier momento, lo cual borra todos los mensajes asociados de nuestra base de datos.
  • El Usuario puede marcar mensajes como leidos y gestionar las conversaciones desde la interfaz de bandeja de entrada.
  • La eliminacion de mensajes de nuestra base de datos no afecta los mensajes en la aplicacion de WhatsApp del Usuario ni del contacto.

15.3 Privacidad de los Mensajes

  • WhatsApp Sender Pro no comparte el contenido de los mensajes con terceros, excepto cuando se utiliza la funcion de IA del chatbot (en cuyo caso los mensajes se envian al proveedor de IA seleccionado por el Usuario).
  • El acceso a los mensajes esta restringido unicamente al Usuario propietario de la cuenta.
  • Los mensajes se almacenan en servidores protegidos con las mismas medidas de seguridad descritas en la seccion de Seguridad de los Datos.
Nota: La funcionalidad de bandeja de entrada requiere que WhatsApp este conectado. Los mensajes recibidos mientras WhatsApp esta desconectado no se capturan en la bandeja de entrada de la Plataforma.
