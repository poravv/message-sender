Inicio Rapido

Sigue estos pasos para enviar tu primer mensaje via API.

1. Obtener token de acceso

curl -X POST /api/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{"apiKey": "TU_API_KEY"}'

2. Enviar un mensaje

curl -X POST /api/v1/messages/send \
  -H "Authorization: Bearer TU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "595991234567",
    "message": "Hola desde la API!"
  }'

3. Envio masivo

curl -X POST /api/v1/messages/bulk \
  -H "Authorization: Bearer TU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "recipients": [
      {"phone": "595991234567", "variables": {"nombre": "Juan"}},
      {"phone": "595981234567", "variables": {"nombre": "Maria"}}
    ],
    "template": "Hola {nombre}, te informamos sobre nuestra promo.",
    "campaignName": "Promo Marzo"
  }'

4. Consultar estado

curl -X GET /api/v1/messages/status/CAMPAIGN_ID \
  -H "Authorization: Bearer TU_TOKEN"

Endpoints Disponibles

Metodo Endpoint Descripcion
POST /api/v1/auth/token Obtener token de acceso
POST /api/v1/messages/send Enviar mensaje individual
POST /api/v1/messages/bulk Envio masivo de mensajes
GET /api/v1/messages/status/:id Estado de campana
POST /api/v1/webhooks Registrar webhook
GET /api/v1/webhooks Listar webhooks
DELETE /api/v1/webhooks Eliminar webhook
